Overview of Task-01: Implement Caesar Cipher
Title

Caesar Cipher Encryption and Decryption using Python

Objective

The aim of this task is to develop a Python program that implements the Caesar Cipher algorithm to encrypt and decrypt text messages. The program should allow users to enter a message and specify a shift value to perform both encryption and decryption.

About Caesar Cipher

The Caesar Cipher is one of the oldest and simplest encryption techniques. It is a substitution cipher in which each letter in the plaintext is shifted by a fixed number of positions in the alphabet.

For example, with a shift value of 3:

Plaintext : ABCXYZ
Ciphertext: DEFABC

Thus, the message:

HELLO

becomes:

KHOOR
Working Principle
The user selects whether to encrypt or decrypt a message.
The user enters the text message.
The user provides a shift value (key).
For encryption, each alphabetic character is shifted forward by the specified value.
For decryption, each alphabetic character is shifted backward by the same value.
Spaces, numbers, and special characters remain unchanged.
Features of the Program
Encrypts plaintext messages.
Decrypts ciphertext messages.
Accepts a user-defined shift value.
Preserves uppercase and lowercase letters.
Ignores non-alphabetic characters.
Technologies Used
Programming Language: Python 3
Development Environment: VS Code / Any Python IDE
Applications
Understanding the fundamentals of cryptography.
Learning basic encryption and decryption concepts.
Educational demonstrations of classical ciphers.
Conclusion

This task provides practical experience in implementing a classical cryptographic technique using Python. Although the Caesar Cipher is not secure for modern communication due to its simplicity, it serves as an excellent introduction to the concepts of encryption, decryption, and information security.
